# 戦え！こうかとん<br>
## 基本機能
* 十字キーで操作
* こうかとんと爆弾が重なると、ゲーム終了
* 爆弾の初期位置はランダム
 
## 実装できた機能一覧<br>
* ゲームオーバー時に画面中央に「GAME OVER」と表示される機能<br>
* SPACEキーを押すと、爆弾を増やせる機能（押した瞬間即死する場合があるのは仕様です）<br>
* Rキーを押すと、爆弾を撃ち落とせる爆弾を発射できる機能<br>
* 時間経過で爆弾が炸裂する機能<br>
* 爆発時の威力（大きさ）にランダム性を持たせた。<br>
* 爆弾の敵味方判別機能（仮）<br>
